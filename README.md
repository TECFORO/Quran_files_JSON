# Quran_files_JSON
this repository is have the Quran verses, and some meanings depending of the languages (or none) in json files.

the sources of the translations are https://tanzil.net/trans/ and https://quranenc.com.

WARNINGS: beware that the translations might not be perfect, and if you have to share these translations, the license imposes that you can't change the translations in themselves; if you have to make an application from these translations, give them their credit; it is not authorized to use their translation into another website without their approval; and if you application or website will gain money from these files, you have to contact them beforehand because they free translations to help the Umah.

the Arabic, Bengali, Italian, Korean, Russian, Thai are from Tanzil.net

Azerbaijani, German, English, Spanish, Persian, French, Hindi, Indonesian, Japanese, Dutch, Portuguese, Albanian, Swahili, Tamil, Telugu, Turkish, Urdu, Uzbek, Vietnamese, and Chinese are from quranenc.com
